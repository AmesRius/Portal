# Local Portal Explorer

> ローカルに散在するWebプロジェクトをブラウザにドラッグ＆ドロップするだけで、検索・起動可能な美しいポータルサイトを瞬時に生成するツール。

PC内に溜まった多数のプロトタイプやモックアップ、過去の制作物が入ったフォルダ群を読み込ませることで、各ディレクトリの `index.html` と `README.md` を自動スキャンし、整理されたダッシュボードとして一覧表示します。

---

## 🚀 主な機能 (Features)

* **高度なディレクトリスキャン**: 親フォルダをドラッグ＆ドロップ（またはファイル選択）するだけで、サブフォルダ内のファイルを再帰的にスキャンします。
* **ローカルサーバー不要の直接起動**: 検出された `index.html` を Blob URL としてメモリ上に展開。ローカルサーバーを立ち上げることなく、ボタン一つで対象のサイトを新規タブで開くことができます。
* **READMEからの自動メタデータ抽出**: 各ディレクトリ内の `README.md` を解析し、マークダウン記法をクリーンアップした上で、プロジェクトの概要テキストとしてカードに自動反映します。
* **瞬時に絞り込めるインクリメンタル検索**: プロジェクト名（フォルダ名）や抽出された説明文を対象とした、高速なリアルタイムテキスト検索機能を搭載。
* **リアルタイム・ステータスボード**: スキャンしたフォルダ数、発見したサイト数、解析したREADMEの数をダッシュボード上部に動的に表示します。
* **こだわりのUIデザイン**: CSSによるノイズテクスチャと走査線（スキャンライン）、モダンなタイポグラフィを組み合わせた、インダストリアルで開発者ライクなインターフェース。

## 🛠 技術スタック (Tech Stack)

外部フレームワークやビルドツールを一切使用せず、モダンブラウザのネイティブAPIをフル活用して構築された単一ファイル構成です。

* **HTML5 / CSS3** (CSS Variables, Grid, Flexbox, Keyframe Animations)
* **Vanilla JavaScript** (ES6+)
* **Web APIs**: 
  * `File API` & `Drag and Drop API` (ディレクトリトラバーサル)
  * `URL.createObjectURL()` (ローカルファイルのBlob URL化)
* **Typography**: [Google Fonts](https://fonts.google.com/) (`IBM Plex Mono`, `Unbounded`)

## 📖 セットアップ・使い方 (Getting Started / Usage)

本ツールはローカルファイルシステムにアクセスするためのサーバー環境などを必要としません。

1. **ツールの起動**
   リポジトリからファイルをダウンロードし、`index.html` をGoogle Chrome, Edge等のモダンブラウザで開きます。
2. **フォルダの読み込み**
   複数のWebサイト（HTMLファイル）が含まれている「親フォルダ」を、画面のドロップゾーンにドラッグ＆ドロップします。
   > **💡 Tip:** 各サブフォルダ内に `index.html` と `README.md` を配置しておくと、ポータル上で最も綺麗に表示されます。
3. **プロジェクトの検索と表示**
   スキャンが完了すると、サイト群がカード形式で一覧表示されます。画面中段の検索バーからキーワードで絞り込んだり、名前順でソートすることが可能です。
4. **サイトを開く**
   各カードの「▶ OPEN SITE」ボタンをクリックすると、ブラウザの新規タブで該当のローカルサイトが即座にプレビュー表示されます。
