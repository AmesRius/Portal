# データコーディング（置換）ツール
> ブラウザ上で完結する、Excel/CSVデータの高速一括置換・クリーニングツール

[![Demo](https://img.shields.io/badge/Demo-Live_View-blue?style=for-the-badge)](https://[デモURLをここに入力])
[![License](https://img.shields.io/badge/License-MIT-green?style=for-the-badge)](LICENSE)

アンケート結果のデータクレンジングや、特定ルールに基づいた文字列の数値化（コーディング）をブラウザ上で簡単に行えるシングルページアプリケーション（SPA）です。データのアップロードからプレビュー、ルールの適用、エクスポートまで、すべての処理をローカル環境（クライアントサイド）で安全かつ高速に実行します。

---

## ✨ 主な機能 (Features)

* **Excel / CSV ファイルの読み込み対応**
    * 複数シートを含むワークブックの読み込みと、シート切り替え機能。
* **リアルタイムなデータプレビュー**
    * 最大2,000行までブラウザ上で軽快に表を描画し、変換前後の状態を即座に確認可能。
* **対象範囲からのユニークデータ抽出**
    * `A2:H12` のようにExcelライクな範囲指定で、置換対象となる重複のない値を自動でリストアップ。
* **直感的な置換ルール設定**
    * UI上から簡単に「置換前 ➡️ 置換後」のルールを追加・編集・削除。
    * 数値に変換可能なデータは自動的に数値型（Number）として処理し、Excel出力時の扱いやすさを向上。
* **セキュアなクライアントサイド処理**
    * サーバーへのデータ送信は一切不要。機密性の高いデータも安全に処理。
* **変換後ファイルのエクスポート**
    * 置換処理を行ったデータを `_coded.xlsx` としてワンクリックでダウンロード。

## 🚀 セットアップ・使い方 (Getting Started / Usage)

本ツールは特別なサーバー環境やビルドプロセスを必要としません。

### 1. 起動方法

リポジトリをクローン、またはダウンロードして、`index.html` をお使いのブラウザ（Chrome, Edge, Firefox, Safariなど）で開くだけで即座に使用可能です。

```bash
git clone [https://github.com/](https://github.com/)[あなたのユーザー名]/[リポジトリ名].git
cd [リポジトリ名]
# macOSの場合
open index.html
# Windowsの場合
start index.html

```

### 2. 操作手順

1. **ファイルの読み込み**: 左パネルの「ファイルの読み込み」から対象のExcel/CSVファイルを選択します（複数シートがある場合はドロップダウンで選択）。
2. **対象範囲と抽出**: プレビュー表を見ながら対象範囲（例: `B2:D100`）を入力し、「範囲から置換対象を抽出」をクリックします。
3. **ルールの設定**: 抽出された値に対して、変換後（例: `よくあてはまる` ➡️ `5`）の値を入力します。「手動でルールを追加」から任意のルールを作ることも可能です。
4. **実行と保存**: 「置換を実行する」ボタンを押すとプレビューが更新されます。確認後、「変換したファイルを保存」からExcelファイルをダウンロードしてください。
