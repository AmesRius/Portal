# UI Tweak Editor

> ローカルのHTML/CSSファイルをブラウザ上で直接プレビュー＆ビジュアル編集。ローカルサーバー不要で動作する、超軽量なフロントエンド開発用インスペクター＆微調整ツール。

開発中のプロトタイプやモックアップのデザイン微調整（Tweak）を、コードエディタとブラウザを行き来することなく、GUI上で直感的に行い、そのCSS差分を即座に取得できるWebアプリケーションです。

---

## 🚀 主な機能 (Features)

* **ローカルフォルダの直接読み込み (サーバーレス)**:
  対象のフォルダを選択するだけで、相対パスでリンクされたCSS・JavaScript・画像ファイルを自動でパース＆インライン化。ローカルサーバーを立ち上げる手間なく、即座にプレビューを描画します。
* **ビジュアル・インスペクター（要素ピッカー）**:
  「要素を選ぶ」モードを有効にし、プレビュー画面上の任意の箇所をクリックするだけで、対象のCSSセレクタパス（DOMツリー構造）を自動解析して取得します。
* **直感的なプロパティ編集パネル**:
  * **スタイル編集**: テキスト（色、サイズ、太さ、行間等）、背景色、枠線、透明度などをスライダーやカラーピッカーでリアルタイムに変更。
  * **ボックスモデル編集**: マージン、パディング、要素の幅・高さの制約（`min-width`, `max-width` 等）を視覚的に調整。
* **レスポンシブ・ビューポート検証**:
  フルスクリーン、PC（1280px）、タブレット（768px）、スマホ（390px）などのビューポートサイズをワンクリックで切り替え可能。任意のピクセル幅の手入力にも対応。
* **CSS差分の自動抽出＆エクスポート**:
  GUI上で変更を加えたスタイルの差分だけを自動的にCSSコードとして生成。「CSS差分をコピー」機能で、お使いのコードエディタにそのまま貼り付け可能です。
* **編集内容のローカル保存**:
  「💾 保存」ボタンを押すと、微調整を加えた状態のHTMLファイルを再構築し、ローカルに即座にダウンロード（エクスポート）できます。

## 🛠 技術スタック (Tech Stack)

外部ライブラリやビルドツール（NPM、Webpack等）への依存は一切ありません。単一のHTMLファイルとモダンブラウザのAPIのみで構築されています。

* **HTML5 / CSS3** (CSS Variables, Flexbox, OSのダークモード設定に連動する `prefers-color-scheme`)
* **Vanilla JavaScript** (ES6+)
* **Web APIs**: 
  * `File API` / `FileReader API` (ローカルファイルの読み込み)
  * `DOMParser` (HTMLの文字列解析とAST操作)
  * `Blob API` / `URL.createObjectURL` (動的ファイルの生成とダウンロード)
  * `Clipboard API` (CSSコードのコピー)

## 📖 セットアップ・使い方 (Getting Started / Usage)

ビルド環境やNode.js等は不要です。

1. **ツールの起動**
   リポジトリからダウンロードした `index.html` を、Google Chrome, Edge等のモダンブラウザで開きます。
2. **プロジェクトの読み込み**
   左上の「フォルダを開く」ボタンをクリックし、調整したいHTMLファイルが含まれるディレクトリを選択します。
3. **ビューポートの設定**
   画面上部の「プレビュー幅」バーから、確認したいデバイスサイズ（PC、スマホ等）を選択します。
4. **要素の選択と編集**
   「🎯 要素を選ぶ」ボタンをクリックし、プレビュー上の任意の要素を選択します。右側のパネル（「スタイル」「ボックス」）を操作してデザインを微調整してください。
5. **コードの適用**
   右パネルの「コード」タブ、または上部の「CSSをコピー」ボタンから、変更した差分コードを取得し、ご自身のソースコードに反映させます。「💾 保存」からHTMLとして書き出すことも可能です。
