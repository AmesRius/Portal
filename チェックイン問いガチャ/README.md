# 問いガチャ — WorkShop Navi

> ワークショップやミーティングのアイスブレイクに最適。ランダムな要素を組み合わせてユニークな「問い」を生成し、参加者の新たな視点や対話を引き出すWebアプリケーションです。

「10年後の自分におすすめの場所」「3日前の知らない人に聞きたいこと」など、予測不能でクリエイティブな質問をワンクリックで作成できます。

---

## 🚀 主な機能 (Features)

* **ランダムな「問い」の自動生成**: 「数字」「間隔」「前後」「対象」「どんな」「何」の6つのセグメントをスロットのように組み合わせ、ユニークな質問文を瞬時に生成します（ガチャ演出アニメーション付き）。
* **柔軟なスロット固定（ロック機能）**: 任意のセグメント（例：「10」「年」「後」「自分」など）を事前に固定してガチャを回すことができ、特定のテーマやワークショップの目的に沿った問いの絞り込みが可能です。
* **タイムトラベル日時計算機能**: 生成された時間条件（例：10年後、5日前、3ヶ月後）に基づき、現在時刻を基準とした「具体的な未来・過去の日時」をリアルタイムで自動計算し、視覚的に表示します。
* **履歴保存と再利用**: 一度生成した問いの履歴を最大20件まで自動保存。リストからクリックするだけで過去の問いと時間データを再表示・復元できます。
* **ワンクリックコピー**: 生成した問いを「コピー」ボタンから即座にクリップボードへ保存。オンライン会議のチャットやMiroなどのホワイトボードツールへの共有がスムーズです。

## 🛠 技術スタック (Tech Stack)

## 📖 セットアップ・使い方 (Getting Started / Usage)

本ツールはローカルサーバーを立ち上げる必要がなく、ブラウザだけで動作します。

1. **起動する**
   リポジトリをダウンロード（またはクローン）し、`index.html` をGoogle Chrome等のモダンブラウザで開きます。
2. **問いを生成する（ガチャを引く）**
   画面中央の「ガチャを引く ✦」ボタンをクリックします。スロットアニメーションの後に、新しい問いが生成されます。
3. **条件を固定する**
   特定の条件で固定したい場合は、画面下部の「🔒 スロットを固定する」パネルから任意の項目（例：対象を「自分」にする等）を選択してからガチャを引いてください。
4. **結果を共有する**
   「📋 コピー」ボタンを押すと、クリップボードにテキストが保存されます。アイスブレイクの場ですぐに共有可能です。
