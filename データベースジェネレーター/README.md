# DB Site Generator

> CSVファイルを読み込ませるだけで、検索・ソート・フィルタリング機能を備えたカスタムデザインのデータベースサイト（単一HTML）を瞬時に生成するツール。

フロントエンドの知識がなくても、お手持ちのデータ（CSV）から、見やすく操作性の高いWebデータベースをブラウザ上で簡単に作成・カスタマイズできるジェネレーターです。

---

## 🚀 主な機能 (Features)

* **CSVファイルの即時読み込み**: ローカルのCSVファイルをブラウザ上で高速パース。1行目を自動的にヘッダー（項目名）として認識します。
* **デザインのリアルタイムプレビュー**: 
  * サイトタイトル、メインカラー、背景色、文字色をカスタマイズ可能。
  * 変更は即座にプレビュー画面に反映され、完成イメージを確認しながら調整できます。
* **高機能な単一HTMLの生成**: 生成されるコードは外部依存のない1つのHTMLファイルにまとまっており、以下の機能を標準搭載しています。
  * **インクリメンタル検索**: 全文を対象としたリアルタイムなキーワード検索。
  * **カラム別フィルタリング**: 列のデータから自動生成されるドロップダウンを使った絞り込み。
  * **データソート機能**: 各列のヘッダーをクリックするだけで昇順・降順の並び替え（数値・文字列の自動判定対応）。

## 🛠 技術スタック (Tech Stack)

ビルド環境や外部ライブラリは一切不要で、モダンブラウザがあればすぐに動作します。

* **Vanilla JavaScript** (ES6+)

## 📖 セットアップ・使い方 (Getting Started / Usage)

本ツールはローカルサーバー不要で、ブラウザのみで動作します。

1. **ツールの起動**
   リポジトリをダウンロードし、`index.html` をお好みのブラウザで開きます。
2. **データのアップロード**
   「1. CSVファイルをアップロード」のエリアから、対象のCSVファイル（1行目がヘッダーのもの）を選択します。
3. **デザインの調整**
   「2. サイト・デザイン設定」で、サイトのタイトルや各種カラー（メイン、背景、文字色）をお好みに合わせて変更します。右側のプレビューウィンドウで実際の見た目を確認できます。
4. **コードの生成と保存**
   「コードを生成する」ボタンをクリックします。画面下部のテキストエリアにHTMLコードが出力されるので、それをコピーして新しい `.html` ファイルとして保存し、サーバー等にアップロードしてご利用ください。
