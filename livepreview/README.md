# 汎用ファイルエディタ & プレビュー (Live File Editor & Preview)

HTML、Markdown、Python、JSON、CSVなどの様々なファイル形式をブラウザ上で編集し、リアルタイムにプレビュー・内容確認ができるポータブルなWebアプリケーションです。

## 🚀 特徴

- **マルチ拡張子対応**: Web（HTML, CSS, JS）、プログラミング（Python, Java, CPP）、データ（CSV, SQL, JSON）、設定（YAML, INI, CONFIG）に加え、新しく **Markdown (.md)** にも対応しました。
- **リアルタイムプレビュー**: HTMLやMarkdownはブラウザ上で即座にレンダリングされます。CSVは表形式（テーブル）、JSONはインデント整形された形式でプレビュー可能です。
- **直感的なUI/UX**: テンプレート読み込み機能、ワンクリックでのクリア、全画面（エディタ最大化）切り替え、ファイル名指定でのローカルダウンロード機能を完備。
- **単一ファイルで完結**: サーバーレスで動作し、`livepreview.html` をブラウザで開くだけですぐに使用可能です。

## 🛠 搭載機能詳細

### 1. エディタ機能
- 拡張子（`.html`, `.md`, `.py`, `.json`, `.csv`, `.sql`, `.yaml`, `.ini`, `.config`）の切り替え。
- 拡張子変更時のデフォルトテンプレート自動読み込み（確認ダイアログ付き）。
- 自由なファイル名設定（ダウンロード時に自動で拡張子を結合）。
- コードの一括クリア機能（ゴミ箱アイコン）。
- エディタ画面の最大化（プレビュー領域の非表示切り替え）。

### 2. プレビュー・検証機能
- **HTML RENDER**: Tailwind CSSが適用された状態で `iframe` 内にリアルタイムレンダリング。
- **MARKDOWN RENDER**: `marked.js` を用いてMarkdownをHTMLへパースし、読みやすい専用スタイルを当てて描画。
- **CSV VIEW**: カンマ区切りのテキストを自動で視認性の高いHTMLテーブルに変換。
- **JSON VIEW**: リアルタイムに構文エラーを検知し、パース成功時はインデント整形して表示。エラー時はエラー箇所を表示。
- **PLAIN TEXT VIEW**: その他のコード・設定ファイルは、XSS対策のエスケープ処理を施した安全なプレーンテキストとして表示。

## 📂 ファイル構成

- `livepreview.html` : アプリケーション本体（HTML / CSS / JavaScript）

## ⚙️ 使用テクノロジー (CDN経由)

- **Tailwind CSS**: 柔軟でモダンなコンポーネントデザイン・レイアウト
- **marked.js**: 高速なMarkdownパース処理

## 📖 使い方

1. `livepreview.html` を任意のウェブブラウザで開きます。
2. エディタ左上のセレクトボックスから、作成・編集したいファイルの拡張子を選択します。
3. 必要に応じてテンプレートを読み込み、エディタにコードを入力します。
4. 右側の「プレビュー / 内容確認」エリアにリアルタイムで結果が表示されます。
5. ファイル名入力欄に任意の名前を入力し、ダウンロードボタンを押すとローカル環境に保存されます。
