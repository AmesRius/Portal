# Text Analytics Editor

> ブラウザ上で完結。リアルタイムな詳細文字数カウント、文字種分析、高度なテキスト整形機能を備えたオールインワンの軽量テキストエディター。

ライティング、編集、データ整理の作業効率を飛躍的に向上させるツールです。文字数の厳密な確認から、原稿用紙換算、不要な空白や改行のクリーンアップまで、執筆に関するあらゆるニーズを一つの画面で完結させます。

---

## 🚀 主な機能 (Features)

* **高度でリアルタイムなカウント機能**
  * **基本カウント**: 全文字数、空白除外、純粋な文字数（改行・空白全除外）、空白数、行数、段落数を即座に算出。
  * **文字種別分析**: ひらがな、カタカナ、漢字、英字、数字、句読点などを自動判別して個別にカウント（折りたたみパネル採用）。
  * **執筆目安の変換**: 約500文字/分での「読了時間」や、400字詰め「原稿用紙」の枚数を自動換算。
* **強力なテキスト整形・クリーンアップツール**
  * 改行の削除、**空白行の削除**、タブの削除。
  * 行頭・行末スペースの削除（Trim）、テキスト内すべてのスペース削除。
  * Markdown記法（`**`, `#`, `- [ ]` など）のワンクリック除去機能。
* **実用的なユーティリティ**
  * 整形操作を取り消す「Undo（戻る）」機能。
  * ワンクリックでの「クリップボードコピー」。
  * 誤操作防止の確認アラート付き「全消去（クリア）」。
  * 作成したテキストを即座にローカルへダウンロードできる「`.txt` で保存」機能。
* **洗練されたレスポンシブUI**
  * [Tailwind CSS](https://tailwindcss.com/)を用いた、モバイルからデスクトップまでシームレスに動作する見やすくモダンなデザイン。

## 🛠 技術スタック (Tech Stack)

ビルドツールやローカルサーバーは一切不要。単一ファイルで軽量かつ高速に動作します。

## 📖 セットアップ・使い方 (Getting Started / Usage)

外部環境への依存がないため、すぐに使い始めることができます。

1. **起動する**
   リポジトリからファイルをダウンロードし、`index.html` をGoogle Chrome, Safari, Edgeなどのモダンブラウザで開きます。
2. **テキストを入力・分析する**
   画面左側（モバイルでは上部）の「エディター」エリアにテキストを入力、またはペーストします。中央パネルに各カウント結果がリアルタイムで表示されます。
3. **テキストを整形する**
   画面右側の「操作ツール」パネルから、適用したい整形オプション（例：「空白行を削除」「行頭・行末スペース削除」など）にチェックを入れ、「実行する」ボタンをクリックします。
4. **出力・保存する**
   整形・推敲が終わったら、ツール下部の「📋 コピー」ボタンでクリップボードへ保存するか、「💾 .txt で保存」ボタンをクリックしてファイルとして書き出します。
