# 教育産業データベース検索・分析ツール (Edu-Industry DB & Analyzer)

> 教育産業の広範な業態を一覧化し、多角的な検索・フィルタリングとリアルタイムなデータ集計をブラウザ上で実現するダッシュボード型データベースツール。

EdTechから伝統的な学校教育、学習支援、教育関連サービスに至るまで、複雑な教育業界のプレイヤーや業態を俯瞰・分析するためのフロントエンドアプリケーションです。

---

## 🚀 主な機能 (Features)

* **インタラクティブな分析ダッシュボード**
  * データの絞り込みに連動してリアルタイムに変化する集計パネル。
  * 「該当レコード数」「B2B / B2C 比率」「提供形態」「中分類ランキング」を視覚的な棒グラフで表示。
* **強力で柔軟な検索・フィルタリング**
  * **全体横断検索**: テーブル内の全データを対象としたインクリメンタルなキーワード検索。
  * **カラム別フィルター（AND検索）**: 項目ごとに最適化されたフィルター（テキスト入力またはドロップダウン選択）を組み合わせた高度な絞り込み。
* **視認性の高いUIとレスポンシブデザイン**
  * B2B、B2C、B2Gなどの属性を直感的なカラーバッジでハイライト。
  * スマートフォンなどの画面幅が狭いデバイスでは、テーブル表示から**カード型UIへと自動で切り替わる**レスポンシブ設計を採用。
* **クライアントサイド完結の超軽量動作**
  * データベース（`data.js`）を読み込むだけで動作。サーバーサイドの処理やビルド環境は一切不要です。

## 🛠 技術スタック (Tech Stack)

外部ライブラリやフレームワークに依存しない、ピュアなWeb標準技術で構築されています。

* **HTML5**
* **CSS3** (CSS Variables, Flexbox/Grid, メディアクエリによるレスポンシブ対応)
* **Vanilla JavaScript** (ES6+, DOM操作, リアルタイム集計ロジック)
* **Data Format**: JavaScript Object Array (`data.js`)

## 📖 セットアップ・使い方 (Getting Started / Usage)

本ツールはローカル環境のブラウザのみで動作します。

1. **プロジェクトの取得**
   リポジトリをクローンするか、ZIPでダウンロードして解凍します。
   ```bash
   git clone [リポジトリURL]

```

2. **ツールの起動**
フォルダ内にある `index.html` を、Google Chrome, Safari, Edgeなどのモダンブラウザで開きます。
3. **データの検索と分析**
* 画面上部のダッシュボードで現在のデータ分布を確認できます。
* 「全体キーワード検索」や、テーブルヘッダー下部にある各項目の入力欄・ドロップダウンを使用して、目的の業態やサービスを絞り込みます（複数条件のAND検索が可能です）。


4. **データのカスタマイズ**
データベースの内容を更新したい場合は、`data.js` 内の `DATABASE` 配列に格納されたJSON形式のオブジェクトを編集・追加するだけで、即座に画面に反映されます。
